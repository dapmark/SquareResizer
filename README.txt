DapLine SquareResizer
=====================
DapLine SquareResizer — утилита для подготовки квадратных изображений. Программа открывает изображение, помещает его в квадратный холст и сохраняет готовый файл рядом с исходником. Размер итогового квадрата выбирается автоматически: 1000x1000 или 1200x1200.

Возможности
-----------
- Открытие изображения через кнопку «Открыть файл».
- Перетаскивание изображения в окно программы.
- Автоматический выбор итогового размера: 1000x1000 или 1200x1200.
- Сохранение результата рядом с исходным файлом.
- Добавление к имени файла суффикса _1000x1000 или _1200x1200.
- Настройка качества сохранения для JPG и WEBP.
- Переключение светлой и тёмной темы.
- Быстрая отправка изображения через меню Windows «Отправить».

Как пользоваться
----------------
1. Запустите DapLine SquareResizer.
2. Откройте изображение кнопкой «Открыть файл» или перетащите его в окно программы.
3. Программа автоматически создаст квадратную версию изображения.
4. Готовый файл будет сохранён рядом с исходным изображением.

Если исходное изображение подходит для размера 1000x1000, результат будет сохранён с суффиксом _1000x1000.
Если изображение крупнее и требуется размер 1200x1200, результат будет сохранён с суффиксом _1200x1200.

Настройки
---------
Настройки хранятся в файле settings.txt. Пример:
quality=92
theme=dark

quality
    Качество сохранения для JPG и WEBP.
    Допустимое значение: от 1 до 100.

theme
    dark — тёмная тема.
    light — светлая тема.
Изменения в settings.txt применяются после перезапуска программы.

Управление качеством
--------------------
Качество можно менять в поле качества внутри программы.
Колесо мыши в поле качества:
- вверх — увеличить значение на 1;
- вниз — уменьшить значение на 1.

Меню Windows «Отправить»
------------------------
Программу можно использовать через меню Windows «Отправить». Для этого добавьте ярлык программы в папку SendTo:
1. Нажмите Win + R.
2. Введите shell:sendto.
3. Нажмите Enter.
4. Поместите туда ярлык DapLine SquareResizer.
После этого изображение можно будет отправлять в программу через контекстное меню Windows.

Поддерживаемые форматы
----------------------
Программа рассчитана на работу с распространёнными форматами изображений, включая JPG, PNG и WEBP.

Системные требования
--------------------
Windows 11 x64. Работа на других версиях не гарантируется.
