DapLine SquareResizer
=====================
DapLine SquareResizer — компактная утилита для подготовки квадратных изображений. Программа открывает изображение, приводит его к квадрату, применяет выбранный размер и сохраняет готовый файл рядом с исходником.

Возможности
-----------
- Открытие изображения через кнопку «Открыть файл».
- Перетаскивание изображения в окно программы.
- Обработка одного или нескольких файлов в обычном режиме.
- Ручной режим с предпросмотром и квадратной кроп-рамкой.
- Управление кроп-рамкой мышью, стрелками, Shift + стрелками, Home и End.
- Умный режим для попытки дорисовать недостающую сторону цветом фона.
- Выбор размера результата: «Авто» или «Обложка».
- Стандартные размеры обложек: 1400x1400, 1200x1200, 1000x1000, 700x700, 600x600, 500x500.
- Настройка качества сохранения для JPG и WEBP.
- Настройка дополнительной резкости после изменения размера.
- Переключение светлой и тёмной темы через settings.txt.
- Сохранение результата рядом с исходным файлом.
- Автоматическое добавление к имени файла суффикса с итоговым размером.
- Быстрая отправка изображения через меню Windows «Отправить».

Как пользоваться
----------------
1. Запустите DapLine SquareResizer.
2. Выберите размер: «Авто» или «Обложка».
3. При необходимости настройте качество, резкость, умный режим или ручной режим.
4. Откройте изображение кнопкой «Открыть файл» или перетащите его в окно программы.
5. В обычном режиме программа сразу сохранит готовый квадратный файл рядом с исходником.
6. В ручном режиме настройте положение кроп-рамки и нажмите «Сохранить».

Режимы размера
--------------
Авто
    Программа делает квадрат на основе размера исходного изображения.
    Если умный режим может безопасно дорисовать фон, квадрат создаётся по большей стороне.
    Если дорисовка невозможна, используется обычное сжатие/растяжение до квадрата.

Обложка
    Программа приводит изображение к ближайшему стандартному размеру обложки:
    1400x1400, 1200x1200, 1000x1000, 700x700, 600x600 или 500x500.
    Минимальный размер в этом режиме — 500x500.

Умный режим
-----------
Умный режим пытается дорисовать недостающую сторону цветом фона, что подходит для почти квадратных изображений с однотонными краями, например для обложек с диском или кругом на белом, чёрном или другом ровном фоне.
Если фон определить нельзя или разница сторон слишком большая, программа не дорисовывает изображение и использует обычное сжатие/растяжение до квадрата.

Ручной режим
------------
Ручной режим нужен для изображений, где автоматическая обработка может испортить важные детали у края. В этом режиме изображение открывается в предпросмотре, а поверх него появляется квадратная кроп-рамка.
Рамку можно двигать:
- мышью;
- стрелками на 1 пиксель;
- Shift + стрелками на 10 пикселей;
- Home — к началу доступного диапазона;
- End — к концу доступного диапазона.
После нажатия «Сохранить» программа берёт выбранный квадрат, применяет выбранный размер и сохраняет результат рядом с исходным файлом. После сохранения предпросмотр очищается, кнопка снова меняется на «Открыть файл».

Резкость
--------
Резкость применяется только после изменения размера.
Доступные варианты:
- Стандартная;
- Повышенная;
- Высокая;
- Максимальная.
Более сильная резкость может подчёркивать детали, но на некоторых изображениях способна усилить артефакты.

Качество
--------
Качество применяется для JPG и WEBP.
Допустимое значение: от 1 до 100.
Качество можно менять в поле внутри программы.
Колесо мыши в поле качества:
- вверх — увеличить значение на 1;
- вниз — уменьшить значение на 1.

Настройки
---------
Настройки хранятся в файле settings.txt рядом с программой. Пример:
quality=92
resize_mode=music_cover
sharp_mode=standard
smart_mode=true
manual_mode=false
theme=dark

quality
    Качество сохранения для JPG и WEBP.
    Допустимое значение: от 1 до 100.

resize_mode
    auto — режим «Авто».
    music_cover — режим «Обложка».

sharp_mode
    standard — стандартная резкость.
    increased — повышенная резкость.
    high — высокая резкость.
    maximum — максимальная резкость.

smart_mode
    true — включить умный режим.
    false — выключить умный режим.

manual_mode
    true — включить ручной режим.
    false — выключить ручной режим.

theme
    light — светлая тема.
    dark — тёмная тема.
Изменения в settings.txt применяются после перезапуска программы.

Имена файлов
------------
Готовый файл сохраняется рядом с исходным изображением.
К имени добавляется суффикс итогового размера, например:
cover.jpg -> cover_1000x1000.jpg
Если файл с таким именем уже существует, программа добавляет номер:
cover_1000x1000_2.jpg

Меню Windows «Отправить»
------------------------
Программу можно использовать через меню Windows «Отправить». Для этого добавьте ярлык программы в папку SendTo:
1. Нажмите Win + R.
2. Введите shell:sendto.
3. Нажмите Enter.
4. Поместите туда ярлык DapLine SquareResizer.
После этого изображение можно будет отправлять в программу через контекстное меню Windows.

Поддерживаемые форматы
----------------------
Программа рассчитана на работу с распространёнными форматами изображений:
JPG, JPEG, PNG, WEBP, BMP, TIF, TIFF.

Системные требования
--------------------
Windows 11 x64. Работа на других версиях Windows не гарантируется.
